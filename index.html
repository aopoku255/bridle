Bridle

A light weight framework

Bridle is a fast, small and light weight framework that makes things like React, Node and Angular event handling easy with an easy to use API that works across a multitude of browsers.

getting started with Bridle

Installation

There are few options to include/import Bridle into your project:

install from npm

You can install Bridle from npm by simply typing...

$ npm install bridle
install from yarn

You can install Bridle from yarn by simply typing...

$ yarn install bridle
install from bower

You can install Bridle from yarn by simply typing...

$ bower install bridle
Use Bridle from CDN

If you want to include bridle in your project, you may use it from CDN. The following files are available:
